# 古宛妮 (Ellie Ku) - Art Designer Portfolio

## 個人作品集 | Personal Portfolio

歡迎來到我的個人作品集網站！這是一個展示我作為藝術設計師的創意作品、技能和經驗的平台。

Welcome to my personal portfolio website! This is a platform showcasing my work as an Art Designer, including my creative projects, skills, and professional experience.

### 📋 Contents | 內容

- **首頁 (Home)**: 個人介紹與整體概覽
- **關於 (About)**: 詳細的自我介紹和職業背景
- **技能 (Skills)**: 設計軟體、核心能力與軟性技能
- **作品 (Projects)**: 代表作品展示
- **經驗 (Experience)**: 工作經驗與教育背景
- **聯繫 (Contact)**: 聯絡方式

### 🎨 Features | 特色

- 響應式設計 (Responsive Design)
- 現代化 UI/UX
- 平滑滾動動畫 (Smooth Scrolling)
- 進入時淡入效果 (Fade-in Animation)

### 🚀 How to Use | 使用方式

1. 開啟 `index.html` 在瀏覽器中
2. 使用導航菜單瀏覽各個部分
3. 點擊「查看作品」按鈕查看項目詳情

### 📱 技術 | Technology

- HTML5
- CSS3
- JavaScript (Vanilla)
- 無依賴 (No Framework Dependencies)
